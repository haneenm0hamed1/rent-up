# Rent App

This is a **Rent Application** built with **React** to practice component-based architecture and frontend development.  
Currently, there is **no live demo**, but the project structure and code showcase the main features and React practices.

## Features
- Browse available rental items
- Add items to a cart (static functionality)
- Responsive design for desktop and mobile
- Built with reusable React components

## Tech Stack
- React
- HTML
- CSS
- JavaScript
